porcupine.images — Load images that come with Porcupine¶

This is a tiny Python module for loading images from porcupine/images. Click that link to view available images on GitHub.

Prefer logo-200x200 over logo if you need to load an image when Porcupine starts. images.get('logo') seems to take about 225 milliseconds on this system while images.get('logo-200x200') returns in about 5 milliseconds.

porcupine.images.get(name)[source]¶

Load a tkinter.PhotoImage from an image file that comes with Porcupine.

The name should be the name of a file in porcupine/images without the extension, e.g. 'triangle'. If this function is called multiple times with the same name, the same image object is returned every time.
