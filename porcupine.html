porcupine — The top level Porcupine package¶

Porcupine’s __init__.py doesn’t contain much code, but it defines these things:

porcupine.__version__¶

The Porcupine version as a human-readable string.

porcupine.version_info¶

A (major, minor, patch) tuple of integers. Use this if you want to check Porcupine’s version.
